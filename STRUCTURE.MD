# 项目根目录结构说明

> 项目：深圳 AI for Human 企业 Agent 挑战赛
> 维护人：冯伟雄

## 文件
- `readme.md`：项目说明文档（项目简介、使用方式等）。
- `start_venv.bat`：Windows 下一键激活 Python 虚拟环境的启动脚本。
- `STRUCTURE.MD`：本文件，记录项目根目录的结构与各文件/子目录用途。

## 子目录
- `.venv/`：Python 虚拟环境目录（由工具自动管理，不手工修改）。
- `.qoder/`：Qoder IDE 的本地工程配置目录（由工具自动维护）。
- `data/`：项目使用到的数据文件、预设文件、样例数据等静态资源。
- `docs/`：开发过程中产生的解释、说明、设计类文档。
- `tests/`：测试代码、debug 调试脚本、临时验证脚本等非生产用途代码。

---
当目录文件发生变化时，请更新 STRUCTURE.MD
