LLM 抽象与 Mock 模式

统一接口层 + 确定性 Mock 实现，评委环境无需任何 API Key

LLMProvider 统一接口
classify_pest(text) 返回 {dimension:P/E/S/T, direction:O/T, severity:H/M/L}
score_value(intel) 返回 {relevance, urgency, authority, scale} 四维评分
summarize(items) 生成 PEST 维度摘要 / Daily 简报执行摘要
gen_swot_strategy() 基于 S/W/O/T 条目生成 SO/ST/WO/WT 四类策略建议
🛡 Mock 模式 (默认)
  • 确定性启发式分类
  • 关键词匹配 + 随机噪声
  • 无需 API Key
  • 毫秒级响应
  • 演示稳定可预期
🚀 真实 LLM (可选)
  • 一键切换配置
  • 兼容 OpenAI API
  • 支持通义千问
  • 支持本地 Ollama
  • 结果更精准

class MockLLMProvider(LLMProvider):
  # 确定性启发式 PEST 分类
  def classify_pest(self, text: str):
    if "tariff" in text.lower():
      return {"dimension": "P",
               "direction": "T",
               "severity": "H"}
    # ... 更多关键词规则
    return self._random_fallback(text)
