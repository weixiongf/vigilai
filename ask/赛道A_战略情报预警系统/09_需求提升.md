
问：

还有一步，我要把通知消息，发送邮件、发到飞书，要怎么接入飞书，怎么把消息推送到飞书个人，如果个人不行就推到群里。

## 一、飞书消息推送的两种方案对比

飞书提供了两条推送消息的路径，适用场景完全不同：

| 对比维度 | 方案一：自定义机器人 Webhook | 方案二：应用 Bot API |
|----------|---------------------------|---------------------|
| **推送方向** | 只能向群聊推送，**无法发给个人** | 可发单聊（个人）+ 群聊 |
| **是否需要审核** | 不需要 | 需要租户管理员审核发布 |
| **是否需要服务端** | 不需要，直接 HTTP POST | 需要后端服务调用 API |
| **配置复杂度** | ★ 极低，5 分钟搞定 | ★★★ 中等 |
| **认证方式** | Webhook URL 内置认证 | App ID + App Secret → token |
| **推荐场景** | 通知某个群，如"战略简报群" | 需要通知指定个人（如各市场负责人） |
| **限制** | 只能在一个群里用 | 需管理员上线应用 |

**建议策略**：
- **黑客松/开发阶段**：用 Webhook 推群聊，最快跑通。
- **正式上线后**：再补应用 Bot API 实现对个人的精准推送。


## 二、方案一：自定义机器人 Webhook（推群聊，推荐先行）

### 2.1 创建机器人并获取 Webhook 地址

1. 在飞书 PC 端，打开目标群聊 → 右上角「···」→「设置」→「群机器人」。在群机器人界面点击「添加机器人」，选择「自定义机器人」。
2. 设置机器人名称（如"战略情报助手"）和描述，点击「添加」。
3. 复制生成的 Webhook 地址，格式为：`https://open.feishu.cn/open-apis/bot/v2/hook/xxxxxxxxxxxxxxxxx`。
4. **安全建议**：在安全设置中勾选「签名校验」，复制签名密钥，后续代码中需要用 HMAC-SHA256 计算签名。

### 2.2 代码实现

```python
# apps/notifications/feishu_webhook.py
import json
import time
import hmac
import hashlib
import base64
import requests
from django.conf import settings


class FeishuWebhookNotifier:
    """飞书自定义机器人 Webhook 推送（仅支持群聊）"""

    def __init__(self):
        self.webhook_url = settings.FEISHU_WEBHOOK_URL
        self.secret = getattr(settings, 'FEISHU_WEBHOOK_SECRET', None)

    def _build_sign(self, timestamp: str) -> str:
        """生成签名（如果配置了签名校验）"""
        if not self.secret:
            return ""
        # 把 timestamp + "\n" + 密钥 作为签名的消息体
        sign_str = f"{timestamp}\n{self.secret}"
        hmac_code = hmac.new(
            sign_str.encode("utf-8"), digestmod=hashlib.sha256
        ).digest()
        return base64.b64encode(hmac_code).decode("utf-8")

    def send_text(self, message: str) -> dict:
        """发送纯文本消息"""
        timestamp = str(int(time.time()))
        sign = self._build_sign(timestamp)

        payload = {
            "timestamp": timestamp,
            "sign": sign,
            "msg_type": "text",
            "content": {"text": message},
        }

        resp = requests.post(
            self.webhook_url,
            json=payload,
            headers={"Content-Type": "application/json; charset=utf-8"},
            timeout=10,
        )
        return resp.json()

    def send_card(self, title: str, elements: list) -> dict:
        """发送富文本卡片消息（适合每日简报摘要）"""
        timestamp = str(int(time.time()))
        sign = self._build_sign(timestamp)

        card = {
            "config": {"wide_screen_mode": True},
            "header": {
                "title": {"tag": "plain_text", "content": title},
                "template": "red" if "高风险" in title else "blue",
            },
            "elements": elements,
        }

        payload = {
            "timestamp": timestamp,
            "sign": sign,
            "msg_type": "interactive",
            "card": card,
        }

        resp = requests.post(
            self.webhook_url,
            json=payload,
            headers={"Content-Type": "application/json; charset=utf-8"},
            timeout=10,
        )
        return resp.json()

    def send_high_impact_alert(self, title: str, dimension: str,
                               market: str, summary: str,
                               action_advice: str, info_url: str) -> dict:
        """发送高风险情报告警卡片"""
        elements = [
            {
                "tag": "div",
                "fields": [
                    {"is_short": True, "text": {"tag": "lark_md",
                                                "content": f"**维度**\n{dimension}"}},
                    {"is_short": True, "text": {"tag": "lark_md",
                                                "content": f"**市场**\n{market}"}},
                ],
            },
            {"tag": "hr"},
            {"tag": "div", "text": {"tag": "lark_md",
                                    "content": f"**摘要**\n{summary}"}},
            {"tag": "div", "text": {"tag": "lark_md",
                                    "content": f"**建议行动**\n{action_advice}"}},
            {
                "tag": "action",
                "actions": [
                    {"tag": "button", "text": {"tag": "plain_text",
                                                "content": "查看详情"},
                     "type": "primary", "url": info_url},
                ],
            },
        ]
        return self.send_card(f"🔴 高风险情报：{title}", elements)
```

### 2.4 飞书个人如何接收群消息？

自定义机器人只能推群聊，但你可以建一个**只有自己一人的群**（飞书允许单人建群），把自定义机器人加进去，就相当于"发给个人"了——这是开发阶段最快捷的做法。


## 三、方案二：应用 Bot API（可推个人，正式上线用）

### 3.1 创建企业自建应用

1. 登录飞书开发者后台（https://open.feishu.cn），点击「创建企业自建应用」。
2. 填写应用名称和描述，点击创建。
3. 在左侧导航栏选择「应用能力」→「添加应用能力」→ 添加「机器人」。
4. 在「凭证与基础信息」页，获取 **App ID** 和 **App Secret**。

### 3.2 配置权限

在「开发配置 → 权限管理」中，至少添加以下权限：

```json
{
  "scopes": {
    "tenant": [
      "im:message:send_as_bot",
      "im:message:readonly",
      "im:chat:read",
      "contact:user.base:readonly"
    ],
    "user": [
      "im:message:readonly"
    ]
  }
}
```

**关键权限说明**：

| 权限 | 作用 |
|------|------|
| `im:message:send_as_bot` | 以机器人身份发送消息（**核心**） |
| `im:message:readonly` | 读取消息 |
| `im:chat:read` | 获取群聊信息（用于发群消息） |
| `contact:user.base:readonly` | 获取用户信息（用于获取 `open_id` 发单聊） |

### 3.3 发布应用

配置完权限后，在「版本管理与发布」中创建版本并提交审核。管理员审核通过后，组织内成员即可在飞书中搜索并使用该应用。

### 3.4 代码实现

```python
# apps/notifications/feishu_bot.py
import json
import time
import requests
from django.conf import settings
from django.core.cache import cache


class FeishuBotAPI:
    """飞书应用 Bot API 推送（支持单聊 + 群聊）"""

    BASE_URL = "https://open.feishu.cn/open-apis"
    TOKEN_CACHE_KEY = "feishu:tenant_access_token"

    def __init__(self):
        self.app_id = settings.FEISHU_APP_ID
        self.app_secret = settings.FEISHU_APP_SECRET

    def _get_tenant_access_token(self) -> str:
        """获取 tenant_access_token（带 Redis 缓存，有效期约 2 小时）"""
        token = cache.get(self.TOKEN_CACHE_KEY)
        if token:
            return token

        url = f"{self.BASE_URL}/auth/v3/tenant_access_token/internal"
        resp = requests.post(
            url,
            json={"app_id": self.app_id, "app_secret": self.app_secret},
            headers={"Content-Type": "application/json; charset=utf-8"},
            timeout=10,
        )
        data = resp.json()
        token = data.get("tenant_access_token", "")
        expires_in = data.get("expire", 7200) - 300  # 提前 5 分钟过期
        cache.set(self.TOKEN_CACHE_KEY, token, timeout=expires_in)
        return token

    def send_text_to_user(self, open_id: str, message: str) -> dict:
        """发送文本消息给指定用户（单聊）"""
        token = self._get_tenant_access_token()
        url = f"{self.BASE_URL}/im/v1/messages"
        params = {"receive_id_type": "open_id"}
        payload = {
            "receive_id": open_id,
            "msg_type": "text",
            "content": json.dumps({"text": message}),
        }
        resp = requests.post(
            url,
            params=params,
            json=payload,
            headers={
                "Authorization": f"Bearer {token}",
                "Content-Type": "application/json; charset=utf-8",
            },
            timeout=10,
        )
        return resp.json()

    def send_text_to_chat(self, chat_id: str, message: str) -> dict:
        """发送文本消息到群聊"""
        token = self._get_tenant_access_token()
        url = f"{self.BASE_URL}/im/v1/messages"
        params = {"receive_id_type": "chat_id"}
        payload = {
            "receive_id": chat_id,
            "msg_type": "text",
            "content": json.dumps({"text": message}),
        }
        resp = requests.post(
            url,
            params=params,
            json=payload,
            headers={
                "Authorization": f"Bearer {token}",
                "Content-Type": "application/json; charset=utf-8",
            },
            timeout=10,
        )
        return resp.json()

    def send_card_to_user(self, open_id: str, card: dict) -> dict:
        """发送卡片消息给指定用户（适合每日简报推送）"""
        token = self._get_tenant_access_token()
        url = f"{self.BASE_URL}/im/v1/messages"
        params = {"receive_id_type": "open_id"}
        payload = {
            "receive_id": open_id,
            "msg_type": "interactive",
            "content": json.dumps(card),
        }
        resp = requests.post(
            url,
            params=params,
            json=payload,
            headers={
                "Authorization": f"Bearer {token}",
                "Content-Type": "application/json; charset=utf-8",
            },
            timeout=10,
        )
        return resp.json()
```

### 3.5 如何获取用户的 open_id？

发送单聊消息需要知道对方的 `open_id`。获取方式：

- **方式一**（推荐）：用户在飞书搜索你的应用名，首次向机器人发送任意消息后，你可以在「事件订阅」中收到回调，从事件体里提取 `sender.sender_id.open_id`。
- **方式二**：调用飞书通讯录 API，通过手机号或邮箱查询用户的 `open_id`。
- **开发阶段快捷方案**：建个只有自己一人的群，用自定义机器人 Webhook 方案（方案一），跳过 open_id 的获取环节。


## 四、Django 统一通知服务层设计

为避免各处代码直接调用飞书/邮件 API，建议封装一个统一的 `NotificationService`：

```python
# apps/notifications/services.py
from typing import Optional
from django.conf import settings
from apps.notifications.feishu_webhook import FeishuWebhookNotifier
from apps.notifications.feishu_bot import FeishuBotAPI
from apps.notifications.email_sender import EmailSender


class NotificationService:
    """统一通知服务：根据配置自动选择推送渠道"""

    def __init__(self):
        self._webhook = None
        self._bot = None
        self._email = None

    @property
    def webhook(self):
        if self._webhook is None:
            self._webhook = FeishuWebhookNotifier()
        return self._webhook

    @property
    def bot(self):
        if self._bot is None:
            self._bot = FeishuBotAPI()
        return self._bot

    @property
    def email(self):
        if self._email is None:
            self._email = EmailSender()
        return self._email

    def notify_high_impact(self, raw_info, channels: Optional[list] = None):
        """高影响情报多渠道推送"""
        if channels is None:
            channels = getattr(settings, 'DEFAULT_NOTIFY_CHANNELS',
                               ['feishu_webhook'])

        results = {}
        for channel in channels:
            if channel == 'feishu_webhook':
                results['feishu_webhook'] = self.webhook.send_high_impact_alert(
                    title=raw_info.title,
                    dimension=raw_info.get_strategic_dimension_display(),
                    market=raw_info.target_market,
                    summary=raw_info.summary or raw_info.content[:200],
                    action_advice="请登录系统查看完整分析与建议",
                    info_url=f"https://your-domain.com/dashboard/market-dynamics/{raw_info.id}/",
                )
            elif channel == 'feishu_bot' and raw_info.assignee_open_id:
                # 如果有指定负责人的 open_id，推单人
                results['feishu_bot'] = self.bot.send_text_to_user(
                    open_id=raw_info.assignee_open_id,
                    message=f"🔔 高风险情报：{raw_info.title}\n\n{raw_info.summary}",
                )
            elif channel == 'email' and raw_info.assignee_email:
                results['email'] = self.email.send_alert(
                    to_email=raw_info.assignee_email,
                    subject=f"🔔 高风险情报：{raw_info.title}",
                    body=self._build_email_body(raw_info),
                )

        return results

    def notify_daily_brief(self, report):
        """每日简报推送"""
        # 推送到飞书群聊
        if getattr(settings, 'FEISHU_WEBHOOK_URL', None):
            self.webhook.send_card(
                title=f"📋 海外市场战略情报日报 ({report.period_start.date()})",
                elements=self._build_brief_card_elements(report),
            )
        # 邮件发送
        # ...

    def _build_email_body(self, raw_info) -> str:
        """构造邮件 HTML 正文"""
        return f"""
        <h2>{raw_info.title}</h2>
        <p><strong>市场：</strong>{raw_info.target_market}</p>
        <p><strong>维度：</strong>{raw_info.get_strategic_dimension_display()}</p>
        <hr>
        <p>{raw_info.summary}</p>
        <p><a href="https://your-domain.com/dashboard/market-dynamics/{raw_info.id}/">查看详情</a></p>
        """

    def _build_brief_card_elements(self, report) -> list:
        """构造简报卡片 elements"""
        # 按格式拼接报告内容
        pass
```

## 五、邮件发送实现（python 内置 smtplib）

### 5.1 代码实现

```python
# apps/notifications/email_sender.py
import smtplib
from email.message import EmailMessage
from django.conf import settings
from typing import Optional


class EmailSender:
    """邮件发送器，使用 python 内置 smtplib"""

    def __init__(self):
        self.host = settings.EMAIL_HOST
        self.port = settings.EMAIL_PORT
        self.username = settings.EMAIL_HOST_USER
        self.password = settings.EMAIL_HOST_PASSWORD
        self.from_email = getattr(settings, 'DEFAULT_FROM_EMAIL', self.username)

    def send_alert(self, to_email: str, subject: str, body: str,
                   cc: Optional[list] = None) -> bool:
        """发送告警邮件"""
        try:
            msg = EmailMessage()
            msg["Subject"] = subject
            msg["From"] = self.from_email
            msg["To"] = to_email
            if cc:
                msg["Cc"] = ", ".join(cc)
            msg.set_content(body, subtype="html", charset="utf-8")

            with smtplib.SMTP(self.host, self.port, timeout=10) as server:
                server.starttls()
                server.login(self.username, self.password)
                server.send_message(msg)
            return True
        except Exception as e:
            print(f"邮件发送失败: {e}")
            return False

    def send_daily_brief(self, to_emails: list, subject: str, html_body: str) -> dict:
        """发送每日简报邮件（批量）"""
        results = {}
        for email in to_emails:
            results[email] = self.send_alert(email, subject, html_body)
        return results
```
