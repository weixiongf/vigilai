```

新框架（战略情报）：
  信息 → 五维分类(竞争/产品/平台/社媒/法规)

       → 战略影响(机会/风险/需关注)
```

### 调整3：数据库模型扩展

在原 `RawInfo` 模型基础上扩展：

```python
class RawInfo(models.Model):
    # ... 原有字段保留 ...
    
    # 新增：战略分析字段
    strategic_dimension = models.CharField(max_length=20, 
        choices=[('competition','竞争'),('product','产品'),('platform','平台'),
                 ('social','社媒'),('regulation','法规')])
    target_market = models.CharField(max_length=50)     # 目标市场
    impact_type = models.CharField(max_length=20, 
        choices=[('opportunity','机会'),('risk','风险'),('watch','需关注')])
    
    # 可追溯性
    source_url = models.URLField()                      # 原始信息链接
    reference_quote = models.TextField()                # 关键引述原文
```
